Blocky McBlockFace

This Chrome Extension blocks ads from known malicious sites. This is done to provide you with a better browsing experience. Normal website functionality will not be effected.

This project is open source.
Github Repository

Extension Options:

Disable: Stop blocking ads across all pages

Whitelist: Do not block ads on the current domain

These settings persist across sessions
